### BhashaQuestV3 - 90-Second Demo Plan

**Objective:** Showcase the core features of the BhashaQuestV3 language learning application in a fast-paced, engaging 90-second demonstration.

**Typical User Flow:** A new user exploring the app for the first time.

---

**(0-10s) Introduction & Home Screen**

*   **Action:** Start the application.
*   **Visual:** The `HomeView` is displayed, showing a clean UI with language options (e.g., Nepali, Kannada).
*   **Narration:** "Welcome to BhashaQuest, a fun and interactive way to learn new languages. From the home screen, I can select a language to start my learning journey."

---

**(10-25s) Starting a Lesson**

*   **Action:** Select a language (e.g., "Nepali") and then click on the first lesson, like "Greetings" or "Numbers".
*   **Visual:** The view transitions from `HomeView` to the `LessonView`. The first exercise appears.
*   **Narration:** "Let's dive into a basic Nepali lesson. The app is designed around interactive exercises to make learning effective and engaging."

---

**(25-65s) Interactive Exercises Showcase**

*   **Action (25-40s):**
    *   Quickly solve a **Multiple Choice Question (MCQ)**.
    *   Click the correct option for a question like "What is 'Hello' in Nepali?".
    *   The app gives instant feedback (correct/incorrect).
*   **Visual:** Show the `MCQExercise` UI. An audio clip of the correct answer plays.
*   **Narration:** "First, we have multiple-choice questions with audio feedback to help with pronunciation."

*   **Action (40-55s):**
    *   Solve a **Tile Order Exercise**.
    *   Drag and drop word tiles to form a sentence like "I am hungry" ("Malai bhok lagyo").
*   **Visual:** Show the `TileOrderExercise` UI. Tiles snapping into place.
*   **Narration:** "Next, we build sentences by ordering word tiles. This helps reinforce grammar and sentence structure."

*   **Action (55-65s):**
    *   Briefly show a **Translation Exercise**.
    *   Type the translation for a simple word or phrase.
*   **Visual:** Show the `TranslateExercise` UI.
*   **Narration:** "We also have translation exercises to test your vocabulary recall."

---

**(65-80s) Lesson Completion & Profile View**

*   **Action:** Complete the final exercise. A lesson summary screen appears. Then, navigate to the `ProfileView`.
*   **Visual:** Show the lesson completion screen (e.g., "Lesson Complete! +50 XP"). Then switch to the `ProfileView`, which displays stats like "Words Learned," "Lessons Completed," and maybe a progress chart.
*   **Narration:** "After each lesson, you get a summary of your performance. All your progress is tracked in your profile, which uses a Spaced Repetition System to schedule future reviews, ensuring you never forget what you've learned."

---

**(80-90s) Conclusion**

*   **Action:** Stay on the `ProfileView`.
*   **Visual:** The profile screen with user stats.
*   **Narration:** "BhashaQuest is built with C++ and the Qt framework, making it a fast and responsive desktop application for everyone. Start your language quest today!"
